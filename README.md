## Description
[cite_start]**push_swap** es un proyecto de algoritmia que consiste en ordenar una pila de números enteros con un conjunto limitado de instrucciones, utilizando el menor número posible de movimientos[cite: 258, 391]. [cite_start]El programa recibe una lista de números y genera la secuencia de operaciones (`sa`, `pb`, `ra`, etc.) necesaria para que queden ordenados de forma ascendente en la pila A[cite: 258, 283].

[cite_start]Este proyecto se centra en la eficiencia y la comprensión de la complejidad algorítmica ($O(n^2)$, $O(n\sqrt{n})$ y $O(n \log n)$), integrando un selector de estrategias que se adapta según el estado inicial de los datos[cite: 279, 328].
